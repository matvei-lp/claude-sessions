# No Abbreviated Variable Names

Не использовать однобуквенные или сокращённые имена переменных (`f`, `v`, `u`, `d`, `c` и т.п.).
Всегда давать полные, описательные имена:
- `f` → `formatted`
- `v` → `value`  
- `u` → `unit`
- `d` → `dose`
- `c` → `counter` / `color` / etc.

This applies to local variables, closure parameters, and any other bindings.
